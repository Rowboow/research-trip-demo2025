⚠️ 注意事項

🚫 絶対に持参禁止

以下のアイテムは絶対に持参しないでください。発見された場合、即座に預かり・処分の対象となります。

持参禁止物品
  • • アルコール類（エタノール系除菌用品は除く）
  • • タバコ・電子タバコ・ライター
  • • 刃物類（カッター・ハサミ等の文具は除く）
  • • ゲーム機・トランプ等の娯楽用品
  • • 過度に高価な装身具・電子機器
  • • 大音量の出る機器（イヤホン使用は可）
行動・マナーについて

集合時間の厳守

全ての集合時間は必ず守ってください。遅刻は他の参加者に迷惑をかけます。

  • • 集合時間の5分前には集合場所に到着すること
  • • 班行動を基本とし、単独行動は禁止
  • • 施設内での写真撮影は許可された場所のみ
  • • 他の利用者への配慮を忘れずに
  • • 体調不良の際は速やかに引率者に報告
宿泊施設でのルール
  • • 消灯時間（22:00）以降は静かに過ごすこと
  • • 他の部屋への出入りは原則禁止
  • • 貴重品は各自で責任を持って管理
  • • 備品の破損・紛失は速やかに報告
  • • 入浴時間を守り、他の利用者に配慮
健康・安全管理
  • • 持病・アレルギーがある場合は事前に詳細を報告
  • • 体調不良を感じたら我慢せずに申し出る
  • • 怪我をした場合は小さなものでも報告
  • • 知らない人についていかない
  • • 緊急時は引率者の指示に従う

📝 課題

📋 研修レポートについて

研修旅行終了後、学習成果をまとめたレポートを提出していただきます。

事前課題
  • • 訪問予定地の歴史・文化について調べる（A4用紙1枚程度）
  • • 研修旅行で学びたいことを3つ以上リストアップ
  • • 班のメンバーと事前打ち合わせを実施
研修中の記録
  • • 毎日の振り返りシートに記入
  • • 見学先での気づきやメモを取る
  • • 班員との討論内容を記録
  • • 印象に残った場面の写真撮影（許可された場所のみ）
事後課題（提出期限：2026年1月15日）
  • • 研修旅行レポート（A4用紙3枚以上、手書き可）
  • • 班別発表資料の作成
  • • 学んだことの感想文（800字以上）
  • • 今後の学習計画への反映
レポート作成のポイント
  • • 単なる感想ではなく、学習した内容を具体的に記述
  • • 事前に調べた内容と実際に見聞きした内容の比較
  • • 班員との議論から得られた新たな視点
  • • 今後の学習や進路選択への影響
  • • 写真や資料を効果的に活用
